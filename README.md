# ❤️ Heart Disease Prediction

A Machine Learning web application that predicts the risk of heart disease based on user input. Built using **Streamlit**, this app provides real-time predictions with an easy-to-use interface.

---

## 🚀 Live Demo  
👉 [Click here to use the app](https://heartdiseaseprediction-xxxx.streamlit.app)

---

## 📌 Features
- 🔍 Predicts heart disease risk instantly  
- 📊 Machine Learning based prediction  
- 🎯 Simple & clean UI  
- ⚡ Fast and real-time results  

---

## 🧠 How It Works
1. User inputs health details (Age, BP, Cholesterol, etc.)  
2. Data preprocessing (Encoding + Scaling)  
3. Model prediction  
4. Output shown as:
   - ⚠️ High Risk  
   - ✅ Low Risk
